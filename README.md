# WriteHer

WriteHer adalah AI Writing Companion berbasis web yang membantu perempuan mengeksplorasi peluang karier di bidang kepenulisan digital.

## Fitur
- Landing page edukatif
- Rekomendasi peluang karier menulis
- Chat AI dengan Gemini API
- Backend Python Flask

## Teknologi
- HTML
- CSS
- JavaScript
- Python
- Flask
- Gemini API
